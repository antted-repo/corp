# 有擎企业系统

有擎官网系统是一套可快速搭建企业官方网站的官网系统，可以通过后台进行简单的配置即可拥有自己的企业系统。

有擎企业系统发布v1.0.0版本，新功能和Bug修复累计5项，积木式搭建网站，页面构建更灵活。

## 系统功能

2019年04月01日有擎企业系统发布v1.0.0版本，更新了以下5项改进：

- [新功能] 清爽的后台管理界面
- [新功能] 前台模块化可定制
- [新功能] 灵活的导航，连接万千页面
- [新功能] 完善的安装文档，更加方便使用
- [新功能] 多平台可部署，让系统更健壮
